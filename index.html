Metafields

Process of adding metafields.

  • Go to 'Settings'

  • Go to 'Metafields'

  • Go to 'Products'

  • Go to 'Add defination'

  • Name: Name to the metafield

  • Namespace and key: Enter the key for the metafield which will be used in the theme.

  • Description: Describe the metafield.

  • Select content type: The conent which can be entered to the field. Different options of it are show above in the image.

Add the metafields to the product as described below.

  • Product subtitle: To add a sub title to the product add this metafield.

  • Short description: If you want to add additional description to the product add this metafield.

  • Size chart: Add a metafield of typt 'Page' with key 'fields.size_chart'. Attach the page which contains the size chart for the related product. This makes it very simple for you to add the size chart on the product page. It is going to be a pop up link on the product page.

  • Page popup: To add a particular pop up page on the product page you can use the metafield.

    • To add a sales timer on any product either on the product page or on the product card you need to add the metafield. Their are 2 different type of sales timers.

  • Sales loop: In sales loop the timer will show a time difference of days set in the metafield with key 'fields.sales_loop'. So create a metafield with the given deifnation.

  • Sales: In this the timer will run till the date set in the metafield of type 'Date' with the key 'fields.sales'. After the time is over the timer will be hidden.

Some common settings

We have some common settings which are available in many sections. So let's see them at one go.

Color scheme

As you have seen in 'theme settings' you have settings for colors there we have 6 different types of color schemes one is primary and 5 other color schemes. Now in the 'color scheme' setting you can select any one of the color schemes for the section or the block.

Screen display

Most of the sections have this setting. Using this 'setting' you can decide whether to show the 'section' or the 'block' only in large screen size or only in small screen size or on both screen sizes. The options are described below:

'Only view on desktop' section is visible only for large screen.

'Only view on mobile' section is visible only for small screen.

'View on both' section is visible on both type of screen.

Desktop and mobile view

Some of the sections like 'About team', 'Brand promo', 'Collection list', 'Featured blog', 'Featured collection', 'Instagram feed', 'Testimonial', 'Text column' use this setting. Using this setting you can decide whether to show a grid or slider on a large screen and on a small screen.

Grid settings

Under grid settings we have the option to select Card/row. In this you can select number of cards per row for desktop view and for small screens their is only 1 card per row.

Slider settings

Under slider settings we have 3 settings.

  • Navigation style: Style for navigation arrows and the navigation bullets.

  • Cards/row: Card per row for large screen. You can also put it in points like 2.5, 3.3

  • Cards/row on small screen: Card per row for small size screen. You can also put it in points like 1.5, 2.3
