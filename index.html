Theme settings

The theme contains some common settings which can be customized in theme settings.

Steps to open 'theme settings':

  • Go to Online Store.

  • Click 'Customize'

  • Click 'Theme settings'

  • Now we are in theme setings

Color

We have a primary color scheme and 5 different color schemes. Primary color scheme is the main color scheme for the website. These color schemes are used at different places. You will get the option to select one of the color scheme.
