Introduction

Theme settings

The theme contains some common settings which can be customized in theme settings.

Steps to open 'theme settings':

  • Go to Online Store.

  • Click 'Customize'

  • Click 'Theme settings'

  • Now we are in theme setings

Color

We have a primary color scheme and 5 different color schemes. Primary color scheme is the main color scheme for the website. These color schemes are used at different places. You will get the option to select one of the color scheme.

Typography

Select a font style for the heading and the text

Social Media

Enter your social media links. These are used at different places in the website.

Favicon

Add a favicon image

Product Compare

To add product compare option then select first option.

Different product comparision options are also given below

Quick View

Settings for product quick view button.

Product Style

In this you have the settings for the color scheme of product sales staus and to show product is new or upcoming on a product card.

Now we will see each settings one by one.

Price Color

  • Normal price color: Color for the orginal price of the product.

  • Price compare color: Color for the compare price of the product.

Sales badge

  • Product sale or off badge: You have 3 options in this. 'Sales' option will show the sales tag. ' Percentage off' this will show the how much Percentage off on a product. 'None' no badge. These are shown below in the images.

    Slaes
    Percentage off
    None

  • Position: You have 4 different positions from which you can select the position of the sales badge.

  • Sale badge color: Text color of the badge.

  • Badge background color: Background color of the badge.

Product Staus

  • Position: You have 4 different positions from which you can select the position of the sales badge.

  • New product badge: Select if you want to show the product status.

  • New product badge color: Text color of the badge.

  • Badge background color: Background color of the badge.

  • Upcoming badge: Select if you want to show product upcoming badge. If a product is out of stock then this badge will be visible.

  • Upcoming badge text: Text for the coming soon badge.

  • Badge background color: Badge background color. Text color will remain same as that of new badge color

Header elements

You have 4 different sections to design your header.

Announcement bar

You have blocks in this section.

Block settings

  • Text area: Write the text here. You can also give links to particular word in the text.

  • Link: Add link to the complete bar.

    Give only one type of link to the bar that is either link to complete bar or link to words only.

  • Auto scroll text: Scroll the text automatically. Check here

    You should use it when the text length is more then the screen size.

  • Screen display: Described above

Section settings

  • Image: Select image that is to be shown.

  • Image ratio: Select height of the image from the available options.

  • Add image animation: If you want to add an animation in the image then select it. To check the animation click here

  • Color scheme: Select a color scheme from the given options.

  • Desktop layout: In this you can decide to keep image first or the text box first.

  • You can put animations on the Image block and text block. This is a scroll animation.

    You can select selct different animations from the list.

  • Screen display: You can decide the visibilty of the section. It have 3 options 'Only view on desktop', 'Only view on mobile' and 'View on both'.

    'Only view on desktop' section is visible only for large screen.

    'Only view on mobile' section is visible only for small screen.

    'View on both' section is visible on both type of screen.
